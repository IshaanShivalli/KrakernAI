# Changelog

All notable changes to this project will be documented in this file.

## v1.0.0

### Added

* Lightweight wake word listener
* Automatic launching of `jarvis.py`
* Prevention of multiple assistant instances
* Background microphone monitoring
* Google Speech Recognition support
* Batch launcher (`run_listener.bat`)
* Automatic Windows startup instructions
* CPU-friendly architecture
* Clean and modular codebase

### Planned

* Offline speech recognition
* Local language model integration
* Plugin architecture
* Memory system
* Custom wake words
* Command history
* Smart home support
* Cross-platform compatibility
* Better NLP
* GUI interface
